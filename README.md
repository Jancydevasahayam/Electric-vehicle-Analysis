# Electric-vehicle-Analysis
Power BI dashboard analyzing EV market trends, adoption rates, charging stations, and state-wise distribution.

## Project Overview
This project focuses on analyzing Electric Vehicle (EV) market trends and adoption patterns using Power BI. The objective is to transform raw EV data into meaningful insights through interactive dashboards and visualizations.

## Tools & Technologies
- Power BI
- Microsoft Excel
- Power Query

## Objectives
- Analyze Electric Vehicle market growth and adoption rates
- Study charging station availability
- Visualize state-wise EV distribution
- Enable data-driven insights through interactive dashboards

## Data Processing
- Cleaned and transformed raw EV datasets using Power Query
- Handled missing values and standardized data formats
- Prepared structured data models for visualization

## Dashboards & Insights
- EV growth trends over time
- State-wise EV adoption analysis
- Charging station distribution
- Interactive slicers and filters for dynamic analysis

## Key Features
- Interactive and user-friendly Power BI dashboards
- Clear visual storytelling using charts and maps
- Efficient data transformation and modeling

## Outcome
The project provides actionable insights into the EV ecosystem, helping understand market trends, regional adoption patterns, and charging infrastructure distribution.
